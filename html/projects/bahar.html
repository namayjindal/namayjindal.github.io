Bahar

Overview

In Feb 2025, I reached out to this company called corner because I was obsessed with their product. They got back to me immediately, and gave me an extremely fun take-home project that involved building a robust web scraping system that would gather information from various sources on the internet (such as Google Maps, Resy, Opentable, OpenStreetMap etc.) and then build a RAG system on top of it. I eventually adapted the project to my won use case by building a UI and also a deployed website. While there's some minor details that still need to be worked out, I decided to call this project 'Bahar' - a word that means both 'outside' and the season 'spring' depending on which syllable you emphasize. These both resonate with me as the point of the project was to help people find the places they could spend time outside at.
